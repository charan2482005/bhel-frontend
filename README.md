# BHEL Frontend

Frontend application for the BHEL HPVP Marketing Management System developed using ReactJS.

## Features

### Authentication
- Login Page
- CAPTCHA Verification
- Session Persistence
- Logout Functionality

### Tender Enquiry Module
- Add Tender Enquiry
- Upload Documents
- Form Validation
- Responsive UI

### View Enquiries
- Search Functionality
- Date-wise Sorting
- Pagination
- Excel Export
- View Documents

### Sale Order Module
- View Sale Orders
- Excel Export

---

## Technology Stack

- ReactJS
- Axios
- CSS
- XLSX
- File Saver

---

## Installation

Clone Repository

```bash
git clone https://github.com/charan2482005/bhel-frontend.git
```

Install Dependencies

```bash
npm install
```

Run Application

```bash
npm start
```

Application URL

```text
http://localhost:3000
```
